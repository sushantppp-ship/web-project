Site Report & Reflection

This report explains my experience of developing this portfolio website. I focused on responsive web design using HTML and CSS, CSS Grid for layouts, and clean semantic markup.

I learned to:

  • Structured a multi-page website using proper semantic HTML5 elements.
  • Created responsive layouts that work well on both desktop and mobile devices.
  • Used CSS Grid and Flexbox to build modern and flexible page layouts.
  • Embedded media such as videos and screenshots to improve visual content.
  • Managed and updated the project using GitHub with regular commits.

Design decisions:

  • Used a red color accent for buttons and highlights to keep the design consistent.
  • Selected a simple sans-serif font to make the content easy to read.
  • Designed a consistent navigation system, including a hamburger menu for mobile screens.
